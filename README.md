## 개요
범용적인 목적을 달성하기 위한 자바스크립트 라이브러리입니다.
한 전역 저장소와 모델-렌더를 기반으로 한 뷰시스템을 갖추고 있으며 다수의 편의 함수로 구성되어 있습니다.
